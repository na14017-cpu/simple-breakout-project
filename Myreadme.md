Simple Breakout

Simple Breakout is a classic retro-style arcade game implemented in C++ using the raylib graphics library. The player controls a paddle to bounce a ball and destroy all blocks in each level. The game features pixel-art graphics, multiple levels, power-ups, collectibles, sound effects, and background music, creating a full and enjoyable retro gaming experience.

The visual theme is inspired by mining and underground settings, with dark earthy tones, glowing portals, and shiny coins. Music and sound effects match this theme, giving a nostalgic arcade feel.
